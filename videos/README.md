## Instructions

1. **Créer les vidéos** avec les noms exacts listés ci-dessus
2. **Placer les fichiers** dans le dossier `/videos/`
3. **Format recommandé**: MP4 (H.264, AAC)
4. **Les formations** chargeront automatiquement les vidéos

### Exemple de structure finale:
```
Formation/
├── videos/
│   ├── 01_essentiel_intro.mp4
│   ├── 02_essentiel_fondamentaux.mp4
│   ├── ... (autres vidéos)
│   └── 12_premium_projet.mp4
├── formations/
│   ├── essentiel.html
│   ├── standard.html
│   └── premium.html
├── assets/
├── index.html
└── dashboard.html
```

## Noms à utiliser exactement

Copie/colle ces noms pour tes vidéos:

- Standard: `01_standard_intro.mp4` ... `08_standard_projet.mp4`
- Premium: `01_premium_intro.mp4` ... `12_premium_projet.mp4`
